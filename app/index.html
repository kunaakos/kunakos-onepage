Pixframe

Though this is only a low-budget prototype of a much larger interactive installation, it is much fun already in its current state.

It's a led matrix displayed in my apartment (or a bar, or a square, or anywhere else), the pattern on which can be controlled from a publicly accessible web app.

It's real-time, it can be built in any size and any resolution, and manipulated by any amount of people at the same time.

To sum it up, it is a fun fusion of doodling, tic-tac-toe and a mood light.

  • HTML5+CSS3+JS
  • node.js
  • Firebase
  • Raspberry Pi

PizzaPhone

For some reason I though it would be great to name the project like this, and it stuck. Sorry, no pizzas.

It's a rotary phone (remember those?) converted into an alarm clock. Tells you the time and current settings if you pick it up and you can dial in new settings.

If you ever used one of these for reals, it'll give you the fuzzy feels and annoy the hell out of you at the same time, as it was made to be as inefficent as possible by today's standards.

Its voice makes you think of TV commercials. Thank hyperdunno for providing it.

  • Arduino
  • AVR C++
